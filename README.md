# mt-condensates
This is the source data accompanying the paper:

"Transcription-dependent phase coexistence of mitochondrial nucleoids and RNA granules"

Plotting and data visualization scripts, image files and raw data are organized by figure in 'Main and Extended Figures'. 
'Supplementary Videos' contains video files.
'mt-condensate scripts' contains the main scripts used for analysis in Fig 1, Fig 2 and Fig 4.
